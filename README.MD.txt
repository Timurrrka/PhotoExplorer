Приложение, использующее API unsplash.com.

Для использования требуется задать дополнительный строковый ресурс api_key для хранения собственного секретного ключа доступа к API. Получить можно здесь https://unsplash.com/oauth/applications

Клиент позволяет просматривать список коллекций и просматривать фото в их составе с возможностью перехода к полноразмерным изображениям. Присутствуют механизмы поиска фотографий и коллекций по ключевым словам. На главной странице отбражается случайная "фотография дня", изменяющаяся ежедневно.

В составе приложения одна активность и четыре переиспользующихся фрагмента.
